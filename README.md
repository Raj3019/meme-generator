## ğŸ”‘ Getting API Keys

To use this app, you need two API keys. Both offer free tiers or credits for new users.

### Groq API Key (For Captions)
1. Go to the [Groq Cloud Console](https://console.groq.com/keys).
2. Sign in with your account.
3. Click on **"Create API Key"**.
4. Copy the key and paste it into the app's setup screen.

### Stability AI Key (For Images)
1. Go to [Stability AI Platform](https://platform.stability.ai/account/keys).
2. Sign up or log in.
3. Navigate to **API Keys** in your account settings.
4. Click **"Create API Key"**.
5. Copy the key and paste it into the app's setup screen.
